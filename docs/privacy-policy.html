Privacy Policy

CountYourMiles is a trip tracking application that works without registration, without ads, and without commercial collection of personal data. All features are accessible without authentication.

1. Local operation and processed data

  • Your trip data (GPS positions, distance, duration, times) is processed only in your browser to display your route, calculate statistics, and generate a local backup file (JSON) if you choose to export your data.
  • This data is never sent to a third-party server by the application itself. It only leaves your device if you manually export a trip file.

2. Geolocation

  • The application uses your browser's geolocation API to track your movements and display your position on the map in real-time.
  • GPS coordinates are neither stored on a remote server nor shared with the site publisher. They are used only for the current session or saved in a local file at your request.

3. Mapping and third-party services

  • Map backgrounds are provided by external services such as OpenStreetMap and Esri, via the Leaflet library. When the map is displayed, your browser makes requests to these services to load map tiles.
  • These requests may transmit technical information (IP address, browser type, etc.) to map providers, according to their own privacy policies.
    • OpenStreetMap: see the OpenStreetMap Privacy Policy for their practices regarding the collection and use of personal data.
    • Esri: see the Esri Privacy Statement for more information on data management when using their mapping services.
  • The application does not include any advertising or behavioral tracking scripts.

4. Hosting on GitHub Pages

  • The site is hosted on GitHub Pages. By accessing the site, certain technical data (IP address, connection logs, etc.) may be collected by GitHub for security, maintenance, or statistical purposes.
  • For more information on the processing of this data, see the GitHub Privacy Statement.

5. Technical features

  • A service worker is used to enable app functionality and speed up site loading. It does not collect or transmit any personal data.
  • Your preferences (language, theme, units) are stored locally in your browser to personalize your experience.
  • You can save your trips as a JSON file. This file is never automatically sent to a server: it remains on your device unless you share it yourself.

6. Your rights

  • You retain control over all your trip data. You can export, delete, or reset the application at any time.
  • Since the site publisher does not have access to any personal data, they cannot respond to requests for access, rectification, or deletion of data.

7. Contact and updates

  • For any questions about privacy, contact the publisher via the project's GitHub page (available by selecting the logo on the site).
